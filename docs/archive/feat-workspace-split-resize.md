# feat: Workspace 文件树支持拖拽调整宽度

- 变更时间：2026-07-17 00:04 (+08:00)

## 变更理由

Agent 对话页的 Workspace 浏览器使用固定宽度文件树，用户无法根据长文件名或预览内容调整左右区域比例。需要让文件树与文件预览之间的竖线可直接左右拖动，同时保持移动端单栏行为不变。

## 变更内容

- `apps/web/components/assistant-ui/workspace-panel.tsx`：将桌面双栏布局改为动态宽度，并增加基于 Pointer Capture 的可拖拽分隔条。
- 拖动和键盘调整时将文件树宽度限制在 180–360px，并根据 Workspace 面板当前宽度为预览区保留至少 220px。
- 分隔条保持 1px 视觉宽度，提供 12px 隐形拖拽热区，以及 hover、拖动和键盘焦点反馈。
- 支持方向键按 16px 调整宽度，Home/End 快速移动到当前边界；移动端仍使用原有单栏文件树/预览切换。
- 未增加轮询、全局常驻监听、依赖或后端接口。
